
# Project Augmented Reality

This README details the usage of the Web App prototype for the Augmented Reality project.




## Authors

- [@Jordana-GC](https://github.com/Jordana-GC)
- [@MichelleCovali](https://github.com/MichelleCovali)


## Overview

This prototype focuses on showing a simulation of how a final product would look like through a Web App.
The prototype achieves this through using the users camera to display on screen a large object, in this case a duck, that the user can then visualise fully by moving their device around the object.


## Features

- 360 object view.
- Seamless device integration.


## Prerequisites

There are no specific prerequisites to run the app as it does nor have to be run locally to function. You can access the live version of the app directly from your browser.

- **Supported Browsers**: 
  - Chrome (latest version)
  - Firefox (latest version)
  - Edge (latest version)
  - Safari (latest version)
